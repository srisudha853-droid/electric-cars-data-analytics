# 🚗 Electric Cars Data Analysis (Tableau)

## 📊 Project Overview
This project focuses on analyzing electric vehicle (EV) data to understand trends, adoption patterns, and key insights using Tableau.

The dashboard provides a clear and interactive view of EV growth, helping in data-driven decision making.

---

## 🛠 Tools Used
- Tableau (Data Visualization)
- Excel / CSV (Dataset)

---

## 📈 Key Insights
- Growth trend of electric vehicles over time
- Distribution of EV types (BEV vs PHEV)
- Analysis by location/region
- Identification of top-performing categories

## ⚙️ Features
- Interactive filters for better analysis
- Visual comparison of EV categories
- User-friendly dashboard design
- Clear representation of trends and patterns

---

## 🎯 Outcome
This project helps in understanding the adoption of electric vehicles ,charging stations in india and supports better decision-making for future planning and analysis.
